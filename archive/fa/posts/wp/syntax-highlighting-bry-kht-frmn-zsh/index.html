Syntax-highlighting برای خط فرمان ZSH

cd ~/.oh-my-zsh/plugins

git clone https://github.com/zsh-users/zsh-syntax-highlighting

و حالا برای فعال کردن آن خط زیر را به ‎.zshrc موجود در پوشه خانگی اضافه می‌کنیم‌:

plugins=(zsh-syntax-highlighting)

و برای اعمال تغییرات:

source ~/.zshrc

حال با تایپ دستورات نتیجه‌ای مانند عک زیر خواهید داشت:

python ./share.py help
